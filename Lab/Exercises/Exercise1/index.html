Text Shadow

text-shadow: [h-shadow] [v-shadow] [color];

Property Values

h-shadow gives the position of the horizontal shadow.
v-shadow gives the position of the vertical shadow.
blur-radius defines the blur radius. Default value is 0.
none removes the shadow.
